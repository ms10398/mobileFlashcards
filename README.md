## Mobile Flashcards App

This project was made under React Nanodegree while learning React Native.

## Setup

Following commands need to be used to start the app.

```
yarn install
yarn start
```

#### This app was tested by me on Android({
  Device: `Moto M`,
  Model: `XT1663`,
  Android: 7.0
})
